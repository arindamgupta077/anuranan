### Add New Video
```
1. Click "+ Add Video" button
2. Paste YouTube URL
3. Click "Auto-fill" (optional)
4. Enter title and description
5. Set display order (0 = first)
6. Choose status (Active/Inactive)
7. Click "Save Video"
```

### Edit Video
```
1. Hover over video thumbnail
2. Click edit (pencil) icon
3. Modify details
4. Click "Save Video"
```

### Toggle Active/Inactive
```
1. Hover over video thumbnail
2. Click toggle icon
3. Status changes instantly
```

### Delete Video
```
1. Hover over video thumbnail
2. Click delete (trash) icon
3. Confirm deletion
```
